The main target of this project is macOS 10.13, as the older version of
this utility fails during some builds.

[GNU sed](https://www.gnu.org/software/sed/) (stream editor) is a non-interactive command-line text editor.

The initial target was macOS 10.13, whose older version of this utility
failed some builds.

**GNU sed** is an open source project hosted on
[Savannah](https://savannah.gnu.org/projects/sed/).
